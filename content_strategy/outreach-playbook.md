# Outreach / Backlink Playbook — CanPay Insights

目标:获得**真实、会传权重的编辑外链**,把 DR 1.1 推上去 → 排名 → 流量。
不是数量游戏。10 条真实相关的链接 > 100 条 guest-post farm 垃圾链接。

---

## ⛔ 绝对不要碰(会害域名,尤其 AdSense 审核期)

- 任何写着 "Submit guest post / Write for us / 付费收录" 的**通用金融 guest post 农场**
  (如 profinanceblog、consultease、homemortgageontario、finanacecareonline 这类)
- Fiverr / 卖链接 / PBN / 链接互换群
- 大量低质目录提交

这些是 Google 明确打击的 link scheme,新站碰了直接进沙盒。

---

## 🎣 你的"可引用资产"(pitch 的钩子)

外链最强的诱饵是**别人没有的原创数据**。你现在有:

1. **最低工资税后研究**(13 省全表)— `/blog/minimum-wage-take-home-pay-canada-2026`
   → 钩子:"Nunavut 工人税后 $34,098 vs Alberta $26,478,全国差 $7,620"
2. **CPP / EI 按薪资全表** — `/blog/how-much-cpp-will-i-pay-2026`、`/blog/how-much-ei-will-i-pay-2026`
3. **CPP2 2026 解释** — `/blog/cpp2-second-additional-cpp-2026`
4. **法语魁北克文章**(给法语站)— `/blog/guide-paie-quebec-2026` 等 5 篇
5. **免费计算器 + iOS app**(竞品少有 app)— `/`、`/link-to-canpay`

记者/博主引用数据时会带链接 = 高质量编辑外链。**先推数据,不要直接求链接。**

## 三条战线(按 ROI)

### 战线 1:记者/媒体引用(最高质量,免费)

注册这两个平台,以"CanPay Insights 创始人 / 加拿大薪资数据"身份回应记者关于个人理财的提问。一次被 Narcity / Daily Hive / MoneySense / CTV 引用 = 抵 100 条目录链接。

- **Featured.com**(原 HARO 后继)— featured.com
- **Qwoted** — qwoted.com

回应时附上你的数据文章链接作为来源。重点回应:最低工资、税后工资、CPP/EI、各省对比类提问。

### 战线 2:资源页 / 工具列表链接(中等,可批量)

让加拿大理财/HR/新移民站把 CanPay 加进他们的"工具 / 资源"列表。**不是 guest post,是请求加一个有用工具链接。**

## 规则

- **个性化**:每封提到对方一篇具体文章(证明你读过)。群发模板一眼看穿 = 删。
- **先给价值**:数据/工具对他们读者有用,不是求链接。
- **一次一封**,不群发。
- **跟进**:第 7 天一句话跟进,第 21 天礼貌收尾,最多两次。
- **真人账号 + 真实邮箱**(你的,不是我代发)。
